Ghost Bookmarker Settings

Configure your Ghost Admin API credentials

The base URL of your Ghost installation (e.g., https://your-ghost-site.com)
Either:
  • Admin API Key (format: keyId:secret) - from Settings → Integrations
  • Staff Token (Personal Access Token) - from Settings → Staff → [Your User] → Personal Tokens

How to get your Ghost API credentials:

Option 1: Admin API Key (Integration)

  1. Go to your Ghost Admin dashboard
  2. Navigate to Settings → Integrations
  3. Click "Add custom integration"
  4. Copy the Admin API key (format: keyId:secret)

Option 2: Staff Token (Personal Access Token) - Recommended

  1. Go to your Ghost Admin dashboard
  2. Navigate to Settings → Staff
  3. Click on your user profile
  4. Scroll to "Personal Tokens" section
  5. Click "Generate Token"
  6. Copy the token (it's a single string, no colon)
